1.行级元素

内容决定元素所占位置，不可以通过css改变宽高

span, em, i, select, a, strong, mark 凡事带有inline的都有文字特性

2.块级元素

独占一行，可以通过css改变宽高

div, p, ul, li, ol, h,

3.行级块元素

内容决定大小，默认不换行，可以改宽高

img, button, input,

4.空元素

br, hr, ..

三大模型：盒模型，层模型， 浮动模型

盒子模型

盒子模型宽度：width + (pading-left + pading-right) + (border-left+border-right) + (margin-left+margin-right)

可视区(一般不包括margin)宽度：width + (pading-left + pading-right) + (border-left+border-right)

body默认有8px 的 margin
left/right/top/bottom常常配合定位出现

层模型---绝对定位

脱离原来位置进行定位，当一个元素是absolute时就会脱离原来的层面，这样就会发生元素叠加现象

relative 定位

保留原来位置进行定位。也会脱离原来的层面，但是原来占有的位置并不会释放

absolute相对于最近的有定位的父级元素进行定位，若没有则相对于文档进行定位

relative相对于原来出生位置进行定位

一般情况用relative来保存位置作参照，用absolute进行定位

BFC -- 块级格式上下文，能改变盒子里的语法规则

如何处发BFC

  1. position:absolute
  2. display:inline-block
  3. float:left/right
  4. overflow:hidden

float模型

浮动可让元素站队，若父元素可容纳则横向排列，若无法容纳则另起一行

浮动元素产生了浮动流，所有产生了浮动流的元素块级元素看不到他们，但还是产生了BFC的元素和文本类属性(inline)以及文本都能看到浮动元素

如果希望父级元素包裹浮动元素-----清除浮动

伪元素--一般针对于元素内部文本，相较于正常元素，伪元素没有元素结构

伪元素天生存在，我们只是通过css将它选中。

伪元素配合content使用。

伪元素存在于任意一个标签里面,常见伪元素：

  1. after
  2. before
  3. first-letter
  4. first-line

当元素被设置为position:absolute或float:left/right内容会自动把元素转换为inline-block

块级元素可以嵌套任何元素，行级元素只能嵌套行级元素

特别的p元素不能嵌套div元素
